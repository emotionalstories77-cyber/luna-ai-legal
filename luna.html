1. Privacy Policy (Zero-Liability Version)

Local-Only Data Governance: Luna is a "Local-First" application. The Developer does not operate servers, databases, or cloud storage; all data (chats and Gemini API Keys) stays exclusively on the user's device.

API Key Security: Keys are stored using hardware-backed encryption (expo-secure-store). The Developer has zero access to these credentials.

SGI Traceability (India IT Rules 2026): In compliance with Rule 3(3), Luna embeds permanent, tamper-resistant metadata and unique provenance markers into all AI-generated content to ensure traceability.
